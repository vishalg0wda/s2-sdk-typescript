# AppendConditionFailed

Failure response message when an Append is aborted due to a failed condition.


## Supported Types

### `errors.One`

```typescript
const value: errors.One = {
  fencingTokenMismatch: "<value>",
};
```

### `errors.Two`

```typescript
const value: errors.Two = {
  seqNumMismatch: 288761,
};
```
